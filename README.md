## Get started

```bash
npx vity
```

## Examples

Name your project

```bash
npx vity my-app
```

Generate your project in the current working directory

```bash
npx vity .
```

Specify a template: react, react-ts, vue, vue-ts, svelte, svelte-ts

```bash
npx vity --template react
```

Use yarn to install packages

```bash
npx vity --with-yarn
```
